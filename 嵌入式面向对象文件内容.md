# 文件说明

## 外设函数文件内容

* `Periph.h`（**变量声明，函数声明**）
  * `include`：
    * 数据类型相关库，不包含外设库
  * 外设结构体声明
    * `typedef struct Periph_desc Periph_desc_t;`
  * **回调函数类型声明**
    * `typedef void (*Periph_func_t)(Periph_desc_t Periph);`
  * 函数声明
    * `void Periph_Init(Periph_Desc_t Periph);`
    * ...
    * **回调函数声明**`void Periph_Callback_Register(Periph_Desc_t* KEY, Periph_Func_t Func);`
* `Periph_desc.h`
  * `include`：
    * 外设库
    * `Periph.h` 
  * 外设结构体定义
    * 包含回调函数功能
* `Periph.c`
  * `include`：
    * `Periph_desc.h` 
  * 函数定义
    * ...
    * **回调函数定义**



## 板级内容

* `board.h`
  * `include`：
    * `Periph_desc.h`
  * 外设实例化声明
    * `extern Periph_desc_t periph;`
  * 始终开启初始化函数声明
    * `void board_lowlevel_init(void);`
* `board.c`
  * 外设实例化定义
  * 始终开启初始化函数定义
  * 中断函数



## main.c

* `include`：
  * `board.h`
* **回调函数功能撰写**
